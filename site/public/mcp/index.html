One catalog of MCP servers, for Claude, Codex, and ChatGPT. Ara-managed connectors fill the gaps that the official ones leave — Resend with hello@ara.so baked in, Railway pre-loaded with our project IDs, and everything else without a hosted MCP yet.

Fastest path

curl -fsSL https://ara.engineer/install | sh && aracli mcp setup-codex

Installs the aracli CLI, then writes every server below into ~/.codex/config.toml. For Claude Desktop and ChatGPT, see the per-client sections further down.

ChatGPT

  1. Open ChatGPT → Settings → Connectors → Create (requires Pro/Business/Enterprise with Developer Mode).
  2. For the Ara MCP: paste https://mcp.ara.engineer/mcp, authenticate with your @ara.so Google account.
  3. For each official server in the list above: either use ChatGPT's built-in gallery (Gmail, Drive, Calendar, Slack all have native ones) or create a custom connector with the URL from the catalog.

Claude (Desktop / Code / web)

  1. Claude web: Settings → Connectors → Add custom connector → paste the URL.
  2. Claude Desktop: Settings → Connectors → Add, or edit ~/Library/Application Support/Claude/claude_desktop_config.json via aracli mcp setup-claude.
  3. Claude Code: edit ~/.claude.json or run claude mcp add.
