ara.engineer

ae — the Ara engineer CLI

Worktrees, dev envs, and agent-friendly skills — one command away.

Install

curl -fsSL https://ara.engineer/install | sh

Requires bun and git. Installs to ~/.bun/bin/ae.

What it does

aehelp + full skill list
ae listall skills (add --json for agents)
ae <skill>print a skill's SKILL.md so an agent can follow it
ae wt [name]spawn an Ara worktree, dev env, cmux layout, and Claude

Flow

ae wt my-feature ← isolated worktree + dev env ae feat ← start a branch (prints instructions) ... code ... ae test ← auto-detect what changed, run tests ae push ← commit, PR, review, merge, deploy

Agent usage

Every skill is discoverable as JSON, so agents running inside any environment can find capabilities at runtime:

ae list --json # catalog ae <skill> --json # metadata + full SKILL.md body
install script github ara.so
